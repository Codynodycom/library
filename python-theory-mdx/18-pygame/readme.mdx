# Урок 1: Введение в Pygame — "Простое окно"

### Цель: Освоить основы Pygame, включая создание окна и базовый игровой цикл.

### Проект: Мини-игра "Простое окно".

**Описание**: В конце урока будет создано окно, где фон изменяет цвет при нажатии клавиш.

#### Содержание:

- Установка Pygame и создание окна.

- Базовый игровой цикл.

- Изменение цвета фона при нажатии клавиш.


# Урок 2: Работа с объектами и координатами — "Бегущий квадрат"

### Цель: Изучить координаты, перемещение объектов и обновление экрана.

### Проект: Мини-игра "Бегущий квадрат".

**Описание**: Квадрат, управляемый игроком, двигается по экрану, оставляя след, когда перемещается.

- Создание объектов с использованием Rect.

- Перемещение квадрата по экрану.

- Обновление экрана и удаление следов объекта.

# Урок 10: Финальный проект — "2D-игра: Приключение героя"

### Цель: Применить все изученные элементы для создания полноценной 2D-игры.

### Проект: Полноценная 2D-игра "Приключение героя".

**Описание**: Игрок управляет героем, который должен преодолевать препятствия, собирать предметы и избегать врагов. Игра использует события клавиатуры для движения, события мыши для стрельбы или взаимодействия с предметами, а также звуковые и текстовые элементы интерфейса.

## Содержание:

- Сборка всех компонентов (спрайты, звуки, интерфейс).

- Создание игрового мира с разными уровнями.

- Настройка сложностей и целей игры.

- Идеи мини-игр

- Простое окно (урок 1): Пример управления фоном.

- Бегущий квадрат (урок 2): Квадрат двигается по экрану, оставляя след.

- Летающая ракета (урок 3): Ракета движется, оставляя след.

- Лабиринт (урок 4): Простое управление персонажем.

- Стрельба по целям (урок 5): Клики по объектам.

- Счётчик кликов (урок 6): Набор очков через клики.

- Поймай падающие объекты (урок 7): Корзина ловит объекты.

- Звуковой лабиринт (урок 8): Лабиринт со звуками.

- Прыгающий мяч (урок 9): Физика и гравитация.
