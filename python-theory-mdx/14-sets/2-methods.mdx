### Применение множеств в реальных задачах

**Проверка уникальности**: Множества автоматически удаляют дубликаты, что делает их удобными для фильтрации уникальных элементов.

```python
numbers = [1, 2, 2, 3, 4, 4, 5]
unique_numbers = set(numbers)
print(unique_numbers)  # {1, 2, 3, 4, 5}
```

***Проверка на принадлежность**: Множества позволяют быстро проверять, содержится ли элемент в коллекции.

```python
if 'apple' in fruits:
    print("Apple is in the set!")
```

**Математические операции**: Множества можно использовать для выполнения множества операций, таких как объединение или пересечение, что полезно в задачах, связанных с обработкой данных.
