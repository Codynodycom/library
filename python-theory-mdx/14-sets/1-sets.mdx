# Множества

**Множества** в Python — это коллекции уникальных элементов, которые не содержат дубликатов. Множества являются неупорядоченными, что означает, что элементы в них не имеют фиксированного порядка, и вы не можете обращаться к ним по индексу.

```python
# Создание множества
fruits = {'apple', 'banana', 'orange'}
print(fruits)  # {'banana', 'orange', 'apple'}
```

Как видно из примера, порядок элементов может отличаться, но все элементы уникальны.

### Зачем использовать множества?

Множества полезны, когда вам нужно хранить уникальные значения или быстро проверять, содержится ли элемент в коллекции. Они также предоставляют множество удобных операций для выполнения математических операций, таких как объединение, пересечение и разность.

### Основные методы работы с множествами

1. `add()`

Метод `add()` добавляет элемент в множество. Если элемент уже существует, он не добавляется, так как множества содержат только уникальные элементы.

```python
fruits.add('mango')
print(fruits)  # {'banana', 'orange', 'apple', 'mango'}

# Попытка добавить дубликат
fruits.add('apple')
print(fruits)  # {'banana', 'orange', 'apple', 'mango'}
```

2. `remove()`

Метод `remove()` удаляет элемент из множества. Если элемента нет, будет вызвано исключение `KeyError`.

```python
fruits.remove('banana')
print(fruits)  # {'orange', 'apple', 'mango'}

# Попытка удалить несуществующий элемент вызовет ошибку
# fruits.remove('pear')  # KeyError: 'pear'
```

3. `discard()`

Метод `discard()` также удаляет элемент из множества, но, в отличие от `remove()`, он не вызывает ошибку, если элемента нет.

```python
fruits.discard('orange')
print(fruits)  # {'apple', 'mango'}

# Попытка удалить несуществующий элемент НЕ вызовет ошибку
fruits.discard('pear')  # Ошибки нет
```

4. `union()`

Метод `union()` возвращает новое множество, содержащее все уникальные элементы из двух или более множеств. Это соответствует математической операции объединения.

```python
tropical_fruits = {'mango', 'pineapple', 'papaya'}
all_fruits = fruits.union(tropical_fruits)
print(all_fruits)
# {'apple', 'mango', 'pineapple', 'papaya'}
```

5. `intersection()`

Метод `intersection()` возвращает множество, содержащее только те элементы, которые присутствуют во всех заданных множествах. Это соответствует математической операции пересечения.

```python
favorite_fruits = {'apple', 'mango', 'strawberry'}
common_fruits = fruits.intersection(favorite_fruits)
print(common_fruits)  # {'apple', 'mango'}
```

6. `difference()`

Метод `difference()` возвращает множество, содержащее элементы, которые находятся в одном множестве, но отсутствуют в другом. Это соответствует математической операции разности множеств.

```python
uncommon_fruits = fruits.difference(favorite_fruits)
print(uncommon_fruits)  # {'mango'}
```
