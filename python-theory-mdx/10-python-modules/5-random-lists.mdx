# Работа со списками с помощью модуля `random`

Модуль `random` в Python предоставляет множество функций, которые помогают работать со списками. С его помощью можно выбирать случайные элементы, перемешивать списки и даже генерировать случайные выборки из них.

### Выбор случайного элемента из списка

Чтобы выбрать случайный элемент из списка, можно использовать функцию `random.choice()`. Эта функция возвращает один случайный элемент из списка.

```python
import random

fruits = ["яблоко", "банан", "вишня", "апельсин", "груша"]
random_fruit = random.choice(fruits)
print("Случайный фрукт:", random_fruit)
```

Этот код выберет случайный фрукт из списка и выведет его.

### Выбор нескольких случайных элементов

Если тебе нужно выбрать несколько случайных элементов из списка, можно использовать функцию `random.sample()`. Она возвращает список из выбранных случайных элементов, при этом элементы не повторяются.

```python
import random

students = ["Алиса", "Боб", "Кейт", "Том", "Лена"]
random_students = random.sample(students, 3)  # Выбираем 3 случайных студента
print("Случайные студенты:", random_students)
```

Этот код выберет трёх случайных студентов из списка.

### Перемешивание элементов списка

Иногда нужно перемешать элементы списка, чтобы они оказались в случайном порядке. Для этого используется функция `random.shuffle()`. Она перемешивает элементы списка на месте, то есть изменяет сам список.

```python
import random

numbers = [1, 2, 3, 4, 5]
random.shuffle(numbers)
print("Перемешанный список:", numbers)
```

Этот код перемешает числа в списке, и они будут выведены в случайном порядке.

### Случайное число из диапазона индексов

Иногда нужно выбрать случайный элемент не по значению, а по его индексу. Для этого можно сгенерировать случайное число в диапазоне индексов списка с помощью `random.randint()` и затем использовать этот индекс для получения элемента списка.

```python
import random

colors = ["красный", "синий", "зелёный", "жёлтый", "оранжевый"]
random_index = random.randint(0, len(colors) - 1)
print("Случайный цвет по индексу:", colors[random_index])
```

Этот код выбирает случайный индекс и выводит цвет, соответствующий этому индексу.

### Генерация списка случайных чисел

Если нужно создать список, заполненный случайными числами, можно воспользоваться циклом и функцией `random.randint()`.

```python
import random

random_numbers = [random.randint(1, 100) for _ in range(5)]
print("Список случайных чисел:", random_numbers)
```

Этот код создаст список из пяти случайных чисел от 1 до 100.

### Удаление случайного элемента из списка

Чтобы удалить случайный элемент из списка, можно сначала выбрать случайный элемент с помощью `random.choice()`, а затем удалить его с помощью метода `remove()`.

```python
import random

animals = ["собака", "кошка", "лошадь", "заяц", "слон"]
random_animal = random.choice(animals)
animals.remove(random_animal)
print(f"Удалённое животное: {random_animal}")
print("Оставшиеся животные:", animals)
```

Этот код удаляет случайно выбранное животное из списка и выводит оставшиеся элементы.
