# Модуль `time` в Python

Модуль `time` в Python позволяет работать со временем. С его помощью мы можем узнать текущее время, измерить продолжительность выполнения программ и даже сделать паузу в коде.

### Что такое время в программировании?

Представь, что ты засек время на секундомере, чтобы узнать, как быстро ты пробежишь 100 метров. Программирование тоже позволяет засекать время, и модуль time – это как секундомер для компьютера. Давай разберемся, как с ним работать.

### Получение текущего времени

Чтобы узнать текущее время в формате секунд с начала "эпохи" (обычно это 1 января 1970 года), используется функция `time.time()`.


```python
import time

current_time = time.time()
print("Текущее время (в секундах с 1970 года):", current_time)
```

Вывод покажет большое число, которое означает количество секунд с начала эпохи. Не очень похоже на часы, но зато полезно для вычислений времени.

### Задержка выполнения программы

Иногда нужно, чтобы программа сделала паузу, например, подождала 5 секунд перед выполнением следующей команды. Для этого есть функция `time.sleep()`.

```python
import time

print("Программа начнется через 3 секунды...")
time.sleep(3)  # Пауза на 3 секунды
print("Поехали!")
```

Эта программа подождет 3 секунды, а затем напечатает сообщение `"Поехали!"`.

### Измерение времени выполнения

Как можно измерить, сколько времени заняло выполнение части программы? Это можно сделать с помощью двух вызовов `time.time()`: сначала перед началом работы, а затем после её завершения.

```python
import time

start_time = time.time()  # Засекаем время до начала
for i in range(1, 1000000):
    pass  # Эта часть программы что-то делает
end_time = time.time()  # Засекаем время после завершения

execution_time = end_time - start_time
print("Время выполнения:", execution_time, "секунд")
```

Этот код покажет, сколько секунд потребовалось для выполнения цикла.

### Форматирование времени

Чтобы получить время в привычном нам виде (часы, минуты, секунды), используется функция `time.strftime()`. Она позволяет форматировать время в нужный формат.

```python
import time

formatted_time = time.strftime("%H:%M:%S")  # Часы:Минуты:Секунды
print("Текущее время:", formatted_time)
```

Этот код покажет текущее время в формате `"часы:минуты:секунды"`, как на часах.

### Время сна и паузы

Еще один полезный трюк с модулем `time` — это создание пауз между действиями. Например, если тебе нужно сделать паузу между выводами текста в консоли.

```python
import time

for i in range(5):
    print(f"Секунда {i + 1}")
    time.sleep(1)  # Пауза на 1 секунду
```

Эта программа будет печатать `"Секунда 1"`, затем подождет 1 секунду, затем `"Секунда 2"`, и так до 5.
