# Модуль `random` в Python: работа с числами

Модуль `random` в Python позволяет генерировать случайные числа, что часто используется в играх, симуляциях и для тестирования. Сегодня мы рассмотрим, как использовать этот модуль для работы с числами, и решим несколько задач.

### Генерация случайных целых чисел

Для генерации случайных целых чисел можно использовать функцию `random.randint(a, b)`, которая возвращает случайное число в диапазоне от `a` до `b` включительно.

```python
import random

random_number = random.randint(1, 100)
print("Случайное число от 1 до 100:", random_number)
```

Этот код выведет случайное число между 1 и 100, которое будет изменяться при каждом запуске программы.

### Генерация случайных дробных чисел

Если тебе нужно сгенерировать число с плавающей точкой (дробное число), ты можешь использовать функцию `random.uniform(a, b)`. Она возвращает случайное дробное число между `a` и `b`.

```python
import random

random_float = random.uniform(0, 1)
print("Случайное дробное число от 0 до 1:", random_float)
```

Этот код сгенерирует дробное число между 0 и 1.

### Генерация случайных чисел от 0 до 1

Функция `random.random()` генерирует случайное число с плавающей точкой в диапазоне от `0` до `1`. Это удобно, например, для симуляций вероятностей.

```python
import random

random_value = random.random()
print("Случайное число от 0 до 1:", random_value)
```

Этот код сгенерирует случайное значение между `0` и `1`.

### Генерация случайных чисел от 0 до 1

Функция `random.random()` генерирует случайное число с плавающей точкой в диапазоне от `0` до `1`. Это удобно, например, для симуляций вероятностей.

```python
import random

random_value = random.random()
print("Случайное число от 0 до 1:", random_value)
```

Этот код сгенерирует случайное значение между `0` и `1`.

### Работа с диапазонами: случайное число с шагом

Иногда нужно сгенерировать случайное число, которое должно следовать определенному шагу. Например, случайное чётное число в диапазоне. Для этого удобно использовать функцию `random.randrange(start, stop, step)`.

```python
import random

random_even = random.randrange(0, 100, 2)
print("Случайное чётное число от 0 до 100:", random_even)
```

Этот код сгенерирует случайное чётное число от `0` до `100`.

### Задача 1: Угадай случайное число

Создай программу, которая загадывает случайное число от `1` до `10`, а пользователь должен угадать его.

```python
import random

secret_number = random.randint(1, 10)
user_guess = int(input("Угадайте число от 1 до 10: "))

if user_guess == secret_number:
    print("Поздравляю, вы угадали!")
else:
    print(f"Не угадали! Загаданное число было {secret_number}.")
```

### Задача 2: Найди остаток

Напиши программу, которая генерирует случайное число от `1` до `100` и выводит остаток от деления этого числа на `7`.

```python
import random

random_number = random.randint(1, 100)
remainder = random_number % 7
print(f"Случайное число: {random_number}, остаток от деления на 7: {remainder}")
```

### Задача 3: Случайные чётные и нечётные числа

Напиши программу, которая генерирует два случайных числа от `1` до `100` и определяет, какое из них чётное, а какое нечётное.

```python
import random

num1 = random.randint(1, 100)
num2 = random.randint(1, 100)

print(f"Число 1: {num1}, {'чётное' if num1 % 2 == 0 else 'нечётное'}")
print(f"Число 2: {num2}, {'чётное' if num2 % 2 == 0 else 'нечётное'}")
```

### Задача 4: Генератор случайного числа с шагом

Создай программу, которая генерирует случайное число с шагом `5` от `0` до `50`.

```python
import random

random_step = random.randrange(0, 51, 5)
print(f"Случайное число с шагом 5: {random_step}")
```

### Задача 5: Игра с угадыванием диапазона

Напиши программу, которая загадывает случайное число от `1` до `100`. Пользователь должен угадать число, и программа сообщает, ближе ли пользователь к загаданному числу (в диапазоне 10 от загаданного числа) или дальше.

```python
import random

secret_number = random.randint(1, 100)
user_guess = int(input("Угадайте число от 1 до 100: "))

if abs(secret_number - user_guess) <= 10:
    print(f"Вы близки! Загаданное число: {secret_number}")
else:
    print(f"Далеко! Загаданное число: {secret_number}")
```
