# Пошаговая инструкция по установке Git на Windows и подключению к GitHub-аккаунту через HTTPS

## Установка Git на Windows

### 1. Скачивание установочного файла
   - Перейдите на официальный сайт Git: [https://git-scm.com/](https://git-scm.com/).
   - На главной странице нажмите на кнопку **"Download for Windows"**, чтобы скачать установочный файл.

### 2. Запуск установочного файла
   - Откройте загруженный установочный файл для начала установки.

### 3. Настройка параметров установки
   - Следуйте указаниям мастера установки. Рекомендуется оставить параметры по умолчанию:
     - **Select Components:** Выберите все компоненты, чтобы установить Git Bash и Git GUI.
     - **Adjusting your PATH environment:** Выберите **"Git from the command line and also from 3rd-party software"**.
     - **Choosing HTTPS transport backend:** Оставьте **"Use the OpenSSL library"**.
     - Остальные параметры оставьте по умолчанию.

### 4. Завершение установки
   - Нажмите **"Install"**, чтобы начать установку. После завершения установки нажмите **"Finish"**.

---

## Настройка Git и подключение к GitHub через HTTPS

### 1. Настройка имени пользователя и email
   - Откройте **Git Bash** (его можно найти через меню "Пуск").
   - Введите следующие команды для настройки имени пользователя и email (замените на свои данные):
     ```bash
     git config --global user.name "Ваше Имя"
     git config --global user.email "ваш.email@example.com"
     ```

### 2. Проверка установки Git
   - Чтобы проверить, что Git установлен правильно, выполните команду:
     ```bash
     git --version
     ```
   - Если установка прошла успешно, вы увидите версию установленного Git.

### 3. Подключение Git к GitHub-аккаунту через HTTPS
   - Для работы с репозиториями на GitHub через HTTPS, вам не нужно настраивать SSH-ключи. Однако необходимо убедиться, что вы сможете аутентифицироваться при работе с репозиториями.

---

## Работа с готовым репозиторием на GitHub через HTTPS

### 1. Клонирование репозитория через HTTPS
   - Откройте нужный репозиторий на GitHub.
   - Нажмите на кнопку **"Code"**, выберите **HTTPS** и скопируйте URL репозитория.
   - Откройте Git Bash и выполните команду клонирования, заменив `URL` на скопированный адрес:
     ```bash
     git clone https://github.com/username/repository-name.git
     ```
   - После этого репозиторий будет скопирован на ваш компьютер.

### 2. Работа с репозиторием
   - Перейдите в папку с клонированным репозиторием:
     ```bash
     cd repository-name
     ```
   - Теперь вы можете работать с файлами репозитория. Примеры команд:
     - **Добавление файла в репозиторий:**
       ```bash
       git add file_name.py
       ```
     - **Создание коммита:**
       ```bash
       git commit -m "Добавлен новый файл"
       ```
     - **Отправка изменений на GitHub:**
       ```bash
       git push origin your-branch
       ```

Теперь Git установлен и настроен, а вы можете работать с репозиториями на GitHub через HTTPS!
