# Вложенные списки

**Вложенные списки** — это списки, которые содержат другие списки как свои элементы. Они позволяют хранить структуры данных с более сложной организацией, например, двумерные массивы, таблицы или матрицы. Каждый элемент вложенного списка может сам быть списком с произвольным количеством элементов.

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix)
# Вывод: 
# [[1, 2, 3], 
#  [4, 5, 6], 
#  [7, 8, 9]]
```

Здесь `matrix` — это вложенный список, состоящий из трёх списков, каждый из которых представляет строку матрицы.

### Как обращаться к элементам вложенных списков

Для доступа к элементам вложенного списка используются два индекса: первый индекс указывает на внешний список, а второй — на элемент во внутреннем списке.

```python
# Получение элемента '5' из второго списка
element = matrix[1][1]
print(element)  # 5
```

В этом примере `matrix[1]` указывает на второй список `[4, 5, 6]`, а `matrix[1][1]` — на элемент `5` во втором списке.

### Изменение элементов во вложенных списках

Вы можете изменять элементы вложенных списков так же, как и элементы обычных списков.

```python
matrix[0][2] = 10
print(matrix)
# Вывод: 
# [[1, 2, 10], 
#  [4, 5, 6], 
#  [7, 8, 9]]
```

Здесь элемент в первой строке и третьем столбце изменился с `3` на `10`.

### Итерация по вложенным спискам

Для работы с вложенными списками можно использовать вложенные циклы.

```python
for row in matrix:
    for element in row:
        print(element, end=" ")
# Вывод: 1 2 10 4 5 6 7 8 9
```

Здесь внешний цикл проходит по каждому внутреннему списку, а внутренний цикл — по элементам каждого списка.

### Применение вложенных списков

- **Математические матрицы**: Вложенные списки часто используются для представления двумерных матриц, что полезно в задачах линейной алгебры, графики и обработки изображений.

- **Таблицы данных**: Вложенные списки могут хранить данные в виде таблицы, где каждая строка представляет отдельную запись с несколькими полями.

```python
students = [
    ['John', 90],
    ['Alice', 85],
    ['Bob', 95]
]

# Вывод всех студентов и их оценок
for student in students:
    print(f'{student[0]}: {student[1]}')
# Вывод:
# John: 90
# Alice: 85
# Bob: 95
```
