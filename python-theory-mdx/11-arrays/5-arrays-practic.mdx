# Массивы на практике

### Нахождение суммы элементов массива

Одна из самых простых и полезных операций с массивами — это нахождение суммы всех элементов.

```python
import array

# Массив целых чисел
numbers = array.array('i', [10, 20, 30, 40, 50])

# Нахождение суммы всех элементов
total = sum(numbers)
print(f'Сумма элементов: {total}')
# Вывод: Сумма элементов: 150
```

### Умножение всех элементов массива на константу

Допустим, вы хотите умножить каждый элемент массива на одно и то же число. Для этого можно использовать цикл.

```python
import array

# Массив целых чисел
numbers = array.array('i', [1, 2, 3, 4, 5])

# Умножение всех элементов на 10
for i in range(len(numbers)):
    numbers[i] *= 10

print(numbers)  # array('i', [10, 20, 30, 40, 50])
```

### Поиск максимального и минимального значения в массиве

Вы можете легко найти максимальное и минимальное значение в массиве с помощью встроенных функций `max()` и `min()`.

```python
import array

# Массив чисел
numbers = array.array('i', [15, 42, 7, 89, 32])

# Поиск максимального и минимального значения
max_value = max(numbers)
min_value = min(numbers)

print(f'Максимум: {max_value}, Минимум: {min_value}')
# Вывод: Максимум: 89, Минимум: 7
```

### Обработка данных с помощью массива (температурные данные)

Представим, что у вас есть массив с температурными данными, и вам нужно вычислить среднюю температуру за неделю.

```python
import array

# Массив температур (градусы Цельсия)
temperatures = array.array('f', [22.5, 24.0, 19.8, 23.7, 21.1, 25.4, 20.6])

# Нахождение средней температуры
average_temperature = sum(temperatures) / len(temperatures)
print(f'Средняя температура за неделю: {average_temperature:.2f}')
# Вывод: Средняя температура за неделю: 22.44
```

### Создание массива символов и работа с ним

Массивы могут также использоваться для работы с символами. Например, массив символов может представлять строку, где каждый символ — отдельный элемент массива.

```python
import array

# Массив символов
chars = array.array('u', ['H', 'e', 'l', 'l', 'o'])

# Объединение символов в строку
word = ''.join(chars)
print(word)  # Вывод: Hello

# Замена символа 'H' на 'h'
chars[0] = 'h'
new_word = ''.join(chars)
print(new_word)  # Вывод: hello
```

### Работа с массивом чисел с плавающей запятой (перевод валют)

Представим, что вы работаете с массивом курсов валют, и вам нужно перевести все значения из долларов в евро (например, по курсу 0.85).

```python
import array

# Массив курсов валют в долларах
exchange_rates_usd = array.array('f', [1.0, 1.2, 1.5, 1.8, 2.0])

# Перевод всех значений в евро
conversion_rate = 0.85
exchange_rates_eur = array.array('f', [rate * conversion_rate for rate in exchange_rates_usd])

print(exchange_rates_eur)
# Вывод: array('f', [0.85, 1.02, 1.275, 1.53, 1.7])
```
