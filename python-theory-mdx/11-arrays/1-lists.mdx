# Списки

**Списки** — это один из самых популярных типов данных в Python. Они представляют собой упорядоченные изменяемые коллекции элементов. Каждый элемент в списке имеет своё место, называемое индексом, начиная с нуля. Списки позволяют хранить элементы разных типов, например, строки, числа и даже другие списки.

- **Упорядоченность**: элементы хранятся в том порядке, в котором вы их добавили.
- **Изменяемость**: вы можете добавлять, удалять и изменять элементы списка.
- **Удобство**: множество встроенных методов упрощают работу с данными в списках.

```python
fruits = ['apple', 'banana', 'cherry']
print(fruits)  # ['apple', 'banana', 'cherry']
```

### Основные методы работы со списками

```python
# append() Добавляет новый элемент в конец списка.
fruits.append('orange')
print(fruits)  # ['apple', 'banana', 'cherry', 'orange']
```

```python
# insert() Вставляет элемент в список на заданную позицию. Остальные элементы смещаются вправо.
fruits.insert(1, 'orange')
print(fruits)  # ['apple', 'orange', 'banana', 'cherry']
```

```python
# remove() Удаляет первое вхождение элемента из списка.
fruits.remove('banana')
print(fruits)  # ['apple', 'cherry']
```

```python
# pop() Удаляет элемент по индексу и возвращает его. Если индекс не указан, удаляется последний элемент.
last_fruit = fruits.pop()
print(fruits)      # ['apple', 'banana']
print(last_fruit)  # 'cherry'
```

```pyhton
# index() Возвращает индекс первого вхождения указанного элемента.
position = fruits.index('banana')
print(position)  # 1
```

```pyhton
# sort() Сортирует элементы списка по возрастанию.
numbers = [3, 1, 2]
numbers.sort()
print(numbers)  # [1, 2, 3]
```
